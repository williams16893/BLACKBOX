How Can We Generate a README file in just  1-click?

BLACKBOX TEAM |  Jun 29, 2023

  1. Download the Blackbox extension on Visual Studio Code (Of course!)

  2. Click on the Extension Button.

  3. Click on the ‘Generate README.md” button.

  4. Select the files you want to be included.

  5. Click “Ok”.

  6. Done!
